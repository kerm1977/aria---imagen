# ARIA - CONVERTER

Convertidor de formatos multimedia y documentos con tema ARIA.

## Características

- **Conversión de Video**: MP4, AVI, MKV, MOV, WebM
- **Conversión de Audio**: MP3, WAV, OGG, FLAC, AAC
- **Extracción de Audio**: Extraer audio de videos
- **Conversión de Imágenes**: JPG, PNG, WebP, GIF, BMP
- **Conversión de Documentos**: DOCX, ODT a PDF
- **Compresión de Imágenes**: Múltiples calidades

## Tema ARIA

- Dark/Light theme toggle
- Barra de título estilo macOS
- Colores vibrantes
- Borde neón animado

## Instalación

```bash
npm install
npm start
```

## Dependencias

- Electron
- fluent-ffmpeg (para video/audio)
- sharp (para imágenes)
- libreoffice-convert (para documentos)
